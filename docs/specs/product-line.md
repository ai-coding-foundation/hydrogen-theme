# Product Line Specification

## Purpose

This spec defines the EVOHERA product line, offer architecture, competitive positioning, and market context. It serves as the single source of truth for product catalog structure, SKU definitions, and go-to-market framing.

---

## Brand Overview

| Attribute        | Value                                                                            |
| ---------------- | -------------------------------------------------------------------------------- |
| **Brand name**   | EVOHERA                                                                          |
| **Tagline**      | One daily drink. Built for your stage.                                           |
| **Category**     | Stage-specific daily supplement drink mix (effervescent sachets in a tin)        |
| **Core promise** | Replace the "7+ pills" routine with one enjoyable, clinically-dosed daily ritual |

## Offer Architecture

| Offer Type                 | Description                        | Notes                                      |
| -------------------------- | ---------------------------------- | ------------------------------------------ |
| **Quarterly subscription** | Best value, default selection      | Primary subscription cadence               |
| **Monthly subscription**   | Monthly replenishment              | Alternative for trial or flexibility       |
| **One-time purchase**      | Single purchase, no recurring      | For non-subscribers                        |
| **Starter Bundle**         | "Find your stage + pick 2 flavors" | Higher AOV, better tasting trial           |
| **90-day commitment**      | "Try for 90 days" framing          | Matches supplement behavior, reduces churn |

---

## Competitive Wedge Options

| Option       | Positioning                                                                                  | Priority    |
| ------------ | -------------------------------------------------------------------------------------------- | ----------- |
| **Option A** | Clinically-dosed minimalist formulas (fewer ingredients, clearer purpose, easier compliance) | Recommended |
| **Option B** | Symptom-led personalization (quiz outputs stage + add-on booster pack later)                 | Alternative |
| **Option C** | Ultra-clean GI-first delivery (gentler minerals, no sweeteners that upset digestion)         | Alternative |

---

## Market Tailwinds

- **Dietary supplements market**: Enormous and still growing; creates space for premium, differentiated DTC brands.
- **Women's health & beauty supplements**: Large segment with continued growth projected.
- **Menopause market**: Projected growth; perimenopause increasingly treated as its own high-intent segment.

---

## Risk & Compliance

| Requirement                   | Implementation                                                                            |
| ----------------------------- | ----------------------------------------------------------------------------------------- |
| Avoid disease claims          | Use "supports" language only; never "treats" or "cures"                                   |
| Substantiate marketing claims | Ensure all claims are substantiated; education focuses on user experience                 |
| Dietary supplement disclaimer | Include standard FDA dietary supplement disclaimer on all product and marketing materials |
| Review system                 | Build review + education system that reinforces "supports" language                       |
