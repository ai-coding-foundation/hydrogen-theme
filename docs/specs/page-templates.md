# Page Templates Specification

## Purpose

This spec defines the required page templates and section modules for the EVOHERA Shopify store. Templates are built on Shopify Online Store 2.0 with custom sections and metaobjects.

---

## Home Template

| #   | Section Module                  | Description                                                         |
| --- | ------------------------------- | ------------------------------------------------------------------- |
| 1   | **Hero**                        | H1 + subcopy + CTA "Find Your Stage"                                |
| 2   | **Trust badges row**            | Doctor formulated, clinically dosed, sugar-free, third-party tested |
| 3   | **Outcomes icons**              | Mood, energy, digestion, sleep                                      |
| 4   | **Stage selector**              | Tabs → product cards → quick add                                    |
| 5   | **Made with safety & efficacy** | Advisor quotes + testing                                            |
| 6   | **Flavor-first section**        | Macro photography                                                   |
| 7   | **Education preview cards**     | Stage Guide + Ingredients + Research                                |
| 8   | **Final CTA + email capture**   |                                                                     |

## Acceptance Criteria

- [ ] Home template includes all 8 section modules in order
- [ ] PDP above-fold: stage tag, benefits, flavor selector, subscription selector (quarterly default), add to cart, trust icons
- [ ] PDP below-fold: timeline, how to use, supplement facts, science callouts, ingredient tabs, comparison table, reviews, FAQ
- [ ] Ingredient Glossary: search + filter by formula; ingredient detail with required fields
- [ ] Stage Guide: timeline, quality icons, advisory board quotes
- [ ] Research Hub: filters by stage + ingredient; internal linking to products and glossary
